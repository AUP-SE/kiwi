I. Overview
The objective of this project is to create a web application
that allows students and teacher to effectively communicate
with each other.

II. Features
    a. a teacher can open a course
    b. a teacher can add students to a course
    c. students can search for a course
    d. students can request to be in a course
    e. teachers can accept or deny a student requesting
       to be in a course
    f. a student can assign themselves as a course helper
    g. a teacher can assign a student to be a tutor
    h. a helper can be in an active or inactive state
    i. a student can declare an absense
    j. a student can compain
    k. a student can give the professor feedback
    l. students can request that a professor opens a class
